# Manual Module

## Purpose

In-app documentation browser for project, technical, and end-user guidance.

## Functionality

- Renders markdown content from `src/manual`.
- Supports nested topics from folder structure.
- Includes full-text search across manual pages.
- Serves images/assets from `src/manual/assets`.

## Content Layout

- `application/`
- `technical/`
- `end-user/`
- `assets/`

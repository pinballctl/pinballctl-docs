# Layout (Playfield) Module

## Purpose

Visual playfield/layout editor for component placement and key/event mapping.

## Functionality

- Drag/drop and resize-style element positioning.
- Component selection and per-component settings.
- Key binding capture and mapped-event linkage.
- Save/load Playfield state and hardware component overlays.

## Key Endpoints

- `GET /api/playfield/state`
- `POST /api/playfield/state`
- `GET /api/playfield/hardware`
